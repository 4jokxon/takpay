# Receipts

Receipts provide a clean proof-of-payment page for customers and merchants.

Receipt URL:

```txt
/receipt/{invoiceId}
```

## Receipt data

A receipt can show:

- Invoice ID
- Amount and currency
- Payment status
- Payer address
- Recipient address
- Transaction hash
- Payment timestamp
- Explorer link

Receipts make merchant reconciliation easier than relying only on a blockchain explorer.
