# Checkout

The checkout page is the customer-facing payment page.

It shows the amount, currency, recipient, invoice status, and wallet actions.

## Main actions

- Connect wallet
- Switch to Arc Testnet
- Pay with USDC
- Submit a transaction hash manually
- Check payment status

## Manual verification

If a customer already paid from another wallet or app, they can paste the transaction hash and ask TakPay to verify it.

This is useful for mobile wallets, external wallet apps, or interrupted checkout sessions.

## Paid state

Once paid, checkout should show a clear success state and link to the receipt.
